# Project Brief

Music streaming app MVP with Android client, Spring Boot backend, and worker pipeline.

Core goals:
- Anonymous browse and playback.
- Optional password or biometric authentication.
- Browse, search, genre filter, playlists, ratings, downloads, uploads.
- Background playback through Media3.
- Backend metadata, auth, search, upload ingestion, HLS URLs, async transcode/subtitle pipeline.
- Offline-first mobile data with Room outbox and sync.
